# Clarity - 人生價值探索 (Life Value Exploration)

## 概述
**Clarity** 是一個協助使用者探索與收斂人生核心價值的網頁應用程式。提供 40 個精選的抽象價值觀詞彙，透過引導式的三階段篩選流程，協助使用者釐清思緒，最終收斂出對自己生命最重要的 5 個核心價值。

## 產品特色 (Product Features)
- **沈浸式體驗**：採用高級深色主題 (Deep Navy) 與玻璃擬態 (Glassmorphism) 設計，搭配流暢的 Framer Motion 動畫，提供專注且優雅的操作體驗。
- **智慧引導流程**：
    - 分段式篩選，降低選擇障礙。
    - 具備智慧跳轉邏輯 (若第一雪段已精準選擇，自動跳過中間收斂步驟)。
- **資料安全暫存**：使用 Session Storage 避免誤觸重新整理導致進度遺失，但隨瀏覽器關閉自動清除，保障隱私。
- **適應性佈局**：桌面版採用 5 欄寬版排列，手機版自動適應，確保閱讀舒適性。

## 技術架構 (Tech Stack)
- **Core**: React + Vite
- **Styling**: Standard CSS (CSS Variables + Flexbox/Grid)
- **Animation**: Framer Motion
- **Deployment**: Zeabur (Static Hosting)

## 操作流程 (User Flow)
使用者透過 3 個階段進行收斂：

1.  **階段一：初步篩選**
    - 從 40 個詞彙中，憑直覺選擇 **10 個以上** 對自己重要的詞彙。
    - *特殊邏輯*：若使用者此階段剛好只選 10 個，系統判定目標明確，將自動跳過階段二，直接進入決選。

2.  **階段二：收斂聚焦**
    - 從階段一的選擇中，進一步刪減保留至 **10 個**。
    - 顯示「自我提問技巧」提示，引導使用者進行價值觀比較。

3.  **階段三：最終決選**
    - 從最後 10 個中，精煉出 **5 個** 最核心的價值。

4.  **結果展示**
    - 展示最終選出的 5 個價值觀。
    - 以精美卡片形式呈現，適合使用者螢幕截圖保存。

## UI/UX 細節需求
1.  **自我提問技巧**：在收斂階段顯示提示：「相較於 A，我覺得 B 更重要，原因是當兩者衝突，而且只能二選一時，我會傾向選擇 B」。
2.  **排版優化**：首頁說明文字需分段清晰；卡片排列在桌面版為 5 欄，確保視覺平衡。
3.  **繁體中文優化**：全站文字需統一使用台灣繁體中文用語。
